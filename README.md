# 🕷️ Stickman Hook
> **Proyecto: Desarrollo de Videojuego 2D/3D**
> **Módulo:** Programación Multimedia y Dispositivos Móviles
> **Curso:** 2º DAM
> **Autores:** Sergiu Alexandru Anghleuta, José Luis González, Roberto Lucas y Marco Sanvoisin

---

## 🎮 Sobre el Proyecto

**Stickman Hook** es un videojuego de plataformas y físicas en 2D. El jugador controla a un intrépido monigote (*stickman*) que debe sortear peligrosos niveles utilizando únicamente un gancho y la fuerza de la inercia para balancearse hasta la meta. 

Este proyecto ha sido desarrollado desde cero, abarcando desde el concepto inicial hasta la compilación final, implementando sistemas de físicas, iluminación, audio, control de escenas y UI.

---

## ⚙️ Instrucciones de Instalación

Para probar el juego, no necesitas instalar el motor de desarrollo. Sigue estos sencillos pasos:

1. Ve a la carpeta `06_Build/` de este repositorio.
2. Descarga el archivo ejecutable de la plataforma correspondiente (Windows/macOS/Linux/Android).
3. **En PC:** Descomprime el archivo (si está en `.zip`) y ejecuta el archivo `.exe` o `.app`.
4. **En Android:** Transfiere el archivo `.apk` a tu dispositivo, permite la instalación de orígenes desconocidos y ejecuta la aplicación.
5. ¡Disfruta del juego!

## 🕹️ Controles del Juego

El juego está diseñado para ser muy intuitivo, utilizando un único botón (*One-Touch*):

* **Ratón (PC) / Pantalla táctil (Móvil):** * **Mantener pulsado (Clic Izquierdo / Tocar):** El personaje lanza el gancho al punto de anclaje más cercano y comienza a balancearse.
  * **Soltar (Levantar el dedo / Soltar Clic):** El personaje suelta la cuerda y sale despedido gracias a la inercia acumulada.
* **Teclado (Opcional PC):**
  * `R`: Reiniciar el nivel actual.
  * `ESC`: Pausar el juego o volver al Menú Principal.
